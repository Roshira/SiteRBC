\doxysubsection{Detailed Description}
Represents the model used for user registration. 

Represents the model used for profile when you login or registrated.

This class contains the necessary properties that a user needs to provide during the registration process. It includes the user\textquotesingle{}s full name, email, password, confirmation of the password, and role.
